# Dog Sprites — Drop Your PNGs Here

Place the 3 static dog sprite images in this folder before building.

| Filename | When shown | Hunger level |
|---|---|---|
| `dog_happy.png` | Happy state | > 60 |
| `dog_neutral.png` | Neutral state | 30 – 60 |
| `dog_sad.png` | Sad / hungry state | < 30 |

## Recommended size
- **240 × 240 px** minimum (displayed at 220×220 in the app)
- PNG with transparent background
- Keep file size under 200 KB each

## Free placeholder sources
- [OpenGameArt.org](https://opengameart.org) — search "dog sprite"
- [itch.io](https://itch.io/game-assets/free) — search "dog pixel art"
- [Kenney.nl](https://kenney.nl/assets) — game-ready free assets

These will be replaced with Pixalot-grade art once the prototype validates the retention loop.
